th:each

th:each를 통해 태그들을 반복 할 수 있다.

이름 나이 성별 주소
홍길동 20 남 서울시 서초구

th:each에 stat을 추가해서 반복상태를 확인 할 수 있다.

index : 0부터 시작하는 인덱스 값
count : 1부터 시작하는 값
current : 현재 객체의 정보
even : 짝수 번째 데이터 여부
odd : 홀수 번째 데이터 여부
first : 첫 번째 데이터 여부
last : 마지막 데이터 여부
